Задание: Разработка интерактивного редактора изображений «ImageTransTool» в MATLAB
Цель: Закрепить навыки работы с изображениями, матрицами, геометрическими преобразованиями и создать интуитивно понятный графический интерфейс пользователя (GUI).

1. Общее описание
Студенту необходимо разработать приложение ImageTransTool, которое позволяет загружать цветные (RGB) изображения, просматривать их и применять к ним последовательность базовых преобразований с визуальным отображением результата. Интерфейс должен быть простым и удобным.

2. Основные функциональные требования
Часть A: Загрузка и базовое отображение
Кнопка «Загрузить изображение»: Открывает стандартный диалог выбора файла. Поддерживаемые форматы: .jpg, .png, .bmp.
Область просмотра: Две панели (или оси — axes) для одновременного отображения исходного (Original) и преобразованного (Result) изображений.

Часть B: Панель преобразований (ядро приложения)
Приложение должно содержать следующие элементы управления для преобразований. Изменение любого параметра должно автоматически обновлять результирующее изображение.

Цветовые преобразования:

Выпадающий список (Dropdown) с вариантами:
RGB (оригинал)
Оттенки серого
Черно-белое (binary)

Гистограмма: Кнопка «Показать гистограмму» для построения гистограммы яркости текущего результирующего изображения (для серого — один график, для RGB — три канала).

Геометрические преобразования:
Поворот:
Ползунок с диапазоном [-180°, +180°] и шагом в 1°. Отображать выбранный угол.
Выпадающий список для выбора метода интерполяции: 'nearest', 'bilinear', 'bicubic'.
Масштабирование (Изменение размера):

Коррекция яркости и контрастности:
Яркость: Ползунок (например, от -100 до +100). Реализация: result = original + brightnessValue.
Контраст: Ползунок (например, множитель от 0.5 до 2.0). Реализация: result = contrastValue * (original - 128) + 128.

Часть C: Управление результатом

Кнопка «Применить»: Текущее результирующее изображение становится новым «исходным» для последующих операций. (Важно для последовательной цепочки преобразований).
Кнопка «Сбросить всё»: Очищает все преобразования, оставляя в «исходном» только последнее загруженное изображение.

4. Пример сценария работы для тестирования
Загрузите цветное изображение «.jpg».
Преобразуйте его в оттенки серого методом.
Увеличьте яркость на +30.
Поверните изображение на 45 градусов с бикубической интерполяцией.
Уменьшите размер изображения в 2 раза с сохранением пропорций.

Загрузите логотип как верхний слой и наложите его с прозрачностью 50%.
